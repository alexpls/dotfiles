# my dotfiles

Activate them with GNU Stow, see: https://dr563105.github.io/blog/manage-dotfiles-with-gnu-stow/

## Key bindings

### Window management

1. Desktop workspaces (aerospace) - `alt-{1-9}`
2. Terminal workspaces (tmux session) - `tmux-leader-{1-9}`
3. Terminal tabs (tmux windows) - `super-{1-9}`
